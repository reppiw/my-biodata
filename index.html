HTML, CSS, Javascript, dan Tailwind

HTML (HyperText Markup Language) adalah tulang punggung setiap website. Fungsinya adalah untuk membuat struktur dan konten halaman web, seperti teks, gambar, dan tautan.

CSS (Cascading Style Sheets) berfungsi untuk memberikan gaya dan tampilan visual pada konten HTML. Ini termasuk warna, font, tata letak (layout), dan desain responsif.

JavaScript (JS) adalah bahasa pemrograman yang membuat website menjadi interaktif dan dinamis. Dengan JS, kita bisa membuat elemen bergerak, memvalidasi formulir, dan memuat data tanpa me-reload halaman.

Tailwind CSS adalah sebuah utility-first CSS framework yang memungkinkan Anda mendesain website dengan cepat langsung di dalam kode HTML Anda menggunakan kelas-kelas siap pakai, tanpa perlu menulis CSS dari nol.

Cara Membuat Website Sederhana

1. Download Software: Download software yang akan digunakan untuk membuat websitenya, disini saya menggunakan Visual Studio Code.

2. Struktur Dasar Web (HTML): Tulis semua konten (teks, judul, tabel, navigasi) menggunakan tag-tag HTML, dan pastikan kerangka dasarnya sudah rapi, termasuk head dan body.

3. Desain Web (CSS): Buat file baru untuk CSS (biodatstyle.css) dan tautkan ke HTML menggunakan "stylesheet". Gunakan selektor CSS (seperti '.section', 'h1', 'table') untuk mengatur tampilan, warna, dan posisi elemen.

4. Interaktivitas (JavaScript): Jika ingin menambahkan interaktivitas (misalnya animasi gelembung partikel), tulis kode di file JavaScript terpisah (biodat.js) dan tautkan ke HTML menggunakan "script".

5. Testing: Buka file HTML di browser Anda dan pastikan semua tautan bekerja dan tampilannya sudah sesuai dengan yg diinginkan.
